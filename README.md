# Una Malan — COM Fabric Yardage Calculator

Calculates Customer's Own Material (COM) fabric yardage for upholstered furniture using industry-standard surface area methods.

## Deploy Your Own

Click the button below to deploy this app for free on Render:

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/atelier-os-ai/una-malan-com-calculator)

## Local Development

```bash
npm install
npm run dev
```

The app runs on `http://localhost:5000`.

## Tech Stack

- **Frontend:** React, Tailwind CSS, shadcn/ui
- **Backend:** Express.js, SQLite (better-sqlite3), Drizzle ORM
- **Engine:** Industry-standard surface area calculation with per-piece-type rules
